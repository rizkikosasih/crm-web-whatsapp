📖 Panduan Integrasi Google Drive API untuk Laravel
✅ LANGKAH 1: Masuk ke Google Cloud Console
Buka: https://console.cloud.google.com/

Login dengan akun Google Anda.

Pilih project yang aktif (contoh: crm-wa-460906)

📌 Jika belum punya project, lakukan langkah berikut untuk membuatnya:

Klik ikon dropdown di bagian atas → New Project

Isi:

Project Name → bebas, misalnya crm-drive

Location/Organization → default atau sesuai kebutuhan

Klik Create

Setelah selesai → klik lagi dropdown → pilih project yang baru dibuat

✅ LANGKAH 2: Aktifkan Google Drive API
Buka: https://console.cloud.google.com/apis/library

Di kolom pencarian, ketik: Google Drive API

Klik hasilnya → Klik tombol Enable

✅ LANGKAH 3: Buat Service Account
Buka: IAM & Admin → Service Accounts
atau langsung ke: https://console.cloud.google.com/iam-admin/serviceaccounts

Klik tombol Create Service Account

Isi formulir:

Service account name: misalnya crm-drive-uploader

Service account ID → otomatis terisi

Klik Create and Continue

Pilih Role (opsional tetapi disarankan) → Editor atau Storage Admin

Klik Done

✅ LANGKAH 4: Buat Kunci (Key) JSON
Temukan Service Account yang sudah dibuat → klik ikon tiga titik → Manage keys

Klik tombol Add Key → Create new key

Pilih tipe: JSON → Klik Create

File JSON akan diunduh otomatis

Ubah nama file menjadi:
google-service-account.json

Pindahkan file tersebut ke dalam Laravel project Anda:
storage/app/public/google-service-account.json

📝 CATATAN PENTING: Mengizinkan Service Account Akses Folder Google Drive
Secara default service account tidak bisa akses folder Google Drive pribadi.
Agar bisa upload, lakukan langkah berikut:

Buat folder baru di Google Drive Anda

Klik kanan pada folder → Share

Masukkan alamat email dari client_email di file JSON
Contoh: crm-drive-uploader@project-id.iam.gserviceaccount.com

Beri akses sebagai: Editor

✅ LANGKAH 5: Mendapatkan Folder ID Google Drive
Buka folder Google Drive di browser

Contoh URL:
https://drive.google.com/drive/folders/1A2B3C4D5E6F7G8H9I

Folder ID adalah bagian akhir URL:
→ 1A2B3C4D5E6F7G8H9I

✅ LANGKAH 6: Tambahkan Konfigurasi ke File .env Laravel
Tambahkan variabel berikut ke file .env:

ini
Salin
Edit
GOOGLE_DRIVE_FOLDER_ID=1A2B3C4D5E6F7G8H9I
🚀 SELESAI!
Sekarang aplikasi Laravel Anda siap melakukan upload file ke folder Google Drive melalui Service Account.
